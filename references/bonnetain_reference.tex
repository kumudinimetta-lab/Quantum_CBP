% The literature reveals a critical unresolved tension. The best
% quantum exact algorithm for knapsack (Helm and May, 2018) achieves
% $O(2^{0.226n})$ but relies on Heuristic~2---an unproven assumption
% about quantum walk update times that has been shown to be
% mathematically unsound in general~\cite{bonnetain2020improved}.
% Furthermore, it requires QRAQM, a hardware model that does not
% currently exist. Simpler quantum approaches based on Grover search
% achieve only $O(2^{n/2})$, offering no improvement over classical
% meet-in-the-middle. This paper addresses this gap by proposing a
% hybrid algorithm that achieves a provable speedup over classical
% meet-in-the-middle using only QRACM memory and without invoking
% any unproven heuristic assumptions, targeting implementability on
% near-term NISQ hardware.
